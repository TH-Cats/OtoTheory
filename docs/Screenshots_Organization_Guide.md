## 🆘 Troubleshooting

### スクリーンショットが受け付けられない
**問題**: "Invalid screenshot dimensions"
**解決**: 
- 正確なピクセルサイズ確認
- Preview.appで画像情報確認
- 必要に応じてリサイズ

### ファイルサイズが大きすぎる
**問題**: "File size exceeds 500MB"
**解決**:
- PNG形式で保存
- ImageOptimなどで最適化
- JPEGに変換（品質90%以上）

### ステータスバーが乱れている
**問題**: 時刻や信号が不適切
**解決**:
- シミュレーターを **⌘K** でステータスバーリセット
- または Simulator → **Features** → **Toggle Status Bar**
